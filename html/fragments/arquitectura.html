Arquitectura de la Aplicación

Estructura de carpetas y archivos recomendada para el proyecto:

tu-proyecto/
├── index.html             <- Página principal (Contenedor)
├── style.css              <- Hoja de estilos principal
├── script.js              <- Lógica principal (navegación, fetch)
│
├── fragments/             <- Piezas de HTML cargadas en 

│   ├── dashboard.html
│   ├── arquitectura.html
│   ├── ejemplos-css.html
│   ├── ejemplos-js.html
│   ├── configuracion.html
│   └── arquitectura.html  <- Este mismo archivo
│
├── examples/              <- Archivos HTML/CSS/JS de ejemplos individuales
│   ├── html/
│   │   └── 01-estructura.html
│   │   └── ... (más ejemplos HTML)
│   ├── css/
│   │   └── ... (ejemplos CSS)
│   └── js/
│       └── ... (ejemplos JS)
│
└── assets/                <- Imágenes, iconos, fuentes (opcional)
    └── images/
        └── logo.png

Esta estructura ayuda a mantener el proyecto organizado a medida que crece.
