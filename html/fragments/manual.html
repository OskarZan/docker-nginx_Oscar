Manual de Actualización del Sitio Web

Este manual te guiará para realizar dos tipos de actualizaciones en el sitio web del repositorio:

  1. Añadir un nuevo ejemplo de código (HTML, CSS, JS) que se mostrará en las secciones "Ejemplos...".
  2. Añadir o modificar una página de información (como "Arquitectura", "Instalación" o una nueva sección) accesible desde el menú lateral.

Tipo 1: Añadir un Nuevo Ejemplo de Código (HTML/CSS/JS)

Sigue estos pasos para agregar una nueva página de ejemplo que los usuarios puedan ver.

Paso 1: Crear el Archivo del Ejemplo

  1. Decide qué tipo de ejemplo es: ¿HTML, CSS o JavaScript?
  2. Navega a la carpeta examples: Dentro de html/.
  3. Elige la subcarpeta correcta:
    • HTML: examples/html/
    • CSS: examples/css/ (o cs/ según tu estructura)
    • JS: examples/js/
  4. (Opcional pero recomendado) Crea una subcarpeta: Para organizar por tema (ej: examples/html/formularios/).
  5. Crea tu archivo .html: Dentro de la carpeta elegida, con el código del ejemplo y un nombre descriptivo (ej: formulario-contacto.html).

Paso 2: Añadir la Tarjeta de Enlace en la Lista Correspondiente

  1. Navega a la carpeta fragments: Dentro de html/.
  2. Abre el archivo .html de la lista:
    • ejemplos-html.html
    • ejemplos-css.html
    • ejemplos-js.html
  3. Busca el contenedor:
    .
  4. Copia y pega una tarjeta existente: El bloque de código completo, como este ejemplo: 
    <div class="ejemplo-card">
    <h3>Navegación</h3>
    <p>Navegación entre páginas en distintas carpetas.</p>
    <a data-url="./examples/html/navega/carpeta1/pagina1.html" data-title="CSS-Tricks: Guía Grid">Ver Ejercicio</a>
</div>
  5. Modifica la tarjeta copiada:
    • Cambia al título de la tarjeta.
    • Escribe una breve descripción.
    • Modifica el atributo data-url para que apunte a la ruta relativa y correcta de tu archivo creado en el Paso 1 (ej: ./examples/html/formularios/formulario-contacto.html). Puedes también cambiar el data-title y el texto del enlace ("Ver Ejercicio").
  1. Guarda el archivo del fragmento (ej: ejemplos-html.html).

¡Listo (Tipo 1)! Al recargar y navegar, verás tu nuevo ejemplo.

Tipo 2: Añadir o Modificar una Página de Información

Sigue estos pasos si quieres añadir una nueva sección informativa o modificar una existente.

Opción A: Modificar una Página de Información Existente

  1. Navega a la carpeta fragments: Dentro de html/.
  2. Busca y abre el archivo .html correspondiente (ej: arquitectura.html, instalacion.html).
  3. Edita el contenido HTML como necesites.
  4. Guarda el archivo. Los cambios se reflejarán al cargar esa sección.

Opción B: Añadir una Nueva Página de Información (Nueva Sección)

  1. Paso 1: Crear el Archivo del Fragmento de Contenido
    1. Navega a html/fragments/.
    2. Crea un nuevo archivo .html (ej: buenas-practicas.html).
    3. Añade el contenido HTML con esta estrctura : 
      <h2 id="main-title">Titulo del fragmento</h2>
<div class="arquitectura-view">
<!-- Aquí va todo el contenido de, por ejemplo "buenas-practicas.html", que aparecerá en la sección main de index.html -->
</div>
    4. Guarda este nuevo archivo.
  2. Paso 2: Añadir un Botón en el Menú Lateral
    1. Abre el archivo principal: html/index.html.
    2. Busca el menú lateral:
      y dentro la lista
        .
      • Copia una línea
        • existente.
      • Modifica el botón copiado:
        • Cambia el texto del botón (ej: "Buenas Prácticas").
        • Cambia el atributo data-fragment para que coincida exactamente con el nombre del archivo creado en el Paso 1 (ej: data-fragment="buenas-practicas.html").

        Ejemplo de botón a añadir en index.html:

      • Guarda el archivo index.html.

¡Listo (Tipo 2)! Al recargar, verás el nuevo botón y al hacer clic cargará tu nueva sección informativa.

Con estos pasos, puedes mantener el sitio web actualizado tanto con nuevos ejemplos prácticos como con información relevante para los usuarios.
